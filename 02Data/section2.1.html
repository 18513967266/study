Web自动化测试课程

WebDriver-元素定位

目标

1. 了解元素各种定位方法
2. 掌握id、name、class_name、tag_name、link_text、partial_link_text定位的使用

1. 为什么要学习元素定位方式？

1. 让程序操作指定元素，就必须先找到此元素；
2. 程序不像人类用眼睛直接定位到元素；

1.1 如何进行元素定位？

html页面由标签构成，标签的基本格式如下：
    <标签名 属性名1="属性值1" 属性名2="属性值2">文本</标签名>
    <input id="userA" type="text" name="userA" placeholder="账号A" />
    <a href="http://www.sina.com.cn" id="fw" target="_blank">新浪</a>
元素定位就是通过元素的信息或元素层级结构来定位元素的。

提示：WebDriver提供了八种定位元素方式

2. WebDriver 元素定位方式

1. id
2. name
3. class_name
4. tag_name
5. link_text
6. partial_link_text
7. Xpath
8. Css

定位方式分类-汇总：

1). id、name、class_name：为元素属性定位
2). tag_name：为元素标签名称
3). link_text、partial_link_text：为超链接定位(a标签)
4). Xpath：为元素路径定位
5). Css：为CSS选择器定位

案例演示环境

1. 为了更好的学习元素定位方式和受限于网络速度的影响，我们案例采用本地的html页面来演示。
   这样可以提高学习效率和脚本执行速率。
2. 语言使用Python
3. 开发工具使用Pycharm
4. selenium使用2.48.0

2.1 id定位

说明：id定位就是通过元素的id属性来定位元素，HTML规定id属性在整个HTML文档中必须是唯一的；
前提：元素有id属性

实现案例-1需求：
    1). 打开注册A.html页面，使用id定位，自动填写(账号A：admin、密码A:123456)
    2). 填写完毕后，3秒钟关闭浏览器窗口

id定位方法

driver.find_element_by_id(id)

id定位实现 步骤分析

1. 导入selenium包 --> from selenium import webdriver
2. 导入time包 --> from time import sleep
3. 实例化火狐浏览器 --> driver=webdriver.Firefox()
4. 打开注册A.html --> driver.get(url)
5. 调用id定位方法 --> driver.find_element_by_id("")
6. 使用send_keys()方法发送数据 --> .send_keys("admin")
7. 暂停3秒 --> sleep(3)
8. 关闭浏览器驱动对象 --> quit()

说明：为了接下来更好的而学习体验，我们先暂时使用下，send_keys()方法，在2.3节元素操作讲解；

id定位 案例-1代码：

from selenium import webdriver
from time import sleep
driver=webdriver.Firefox()
url='E:\\测试\\课件\\Web自动化\\Web自动化课件\\02img\\注册A.html'
driver.get(url)
user=driver.find_element_by_id("userA")
user.send_keys("admin")
pwd=driver.find_element_by_id("passwordA")
pwd.send_keys("123456")
sleep(3)
driver.quit()

id定位-总结

1. 导包
2. url中\\转义
3. id定位方法
4. 发送内容方法
5. 暂停方法
6. 关闭浏览器驱动对象

前提：元素有name属性

2.4 tag_name定位

说明：tag_name定位就是通过标签名来定位；
      HTML本质就是由不同的tag(标签)组成，而每个tag都是指同一类，所以tag定位效率低，一般不建议使用；
实现案例-1需求：
    1). 打开注册A.html页面，使用tag_name定位，自动填写(账号A：admin)
    2). 填写完毕后，3秒钟关闭浏览器窗口

tag_name定位方法

driver.find_element_by_tag_name(tag_name)

2.6 partial_link_text定位

说明：partial_link_text定位是对link_text定位的补充，partial_link_text为模糊匹配；link_text全部匹配

    2). 返回的定位元素格式为数组(列表)格式；
说明：
    1). 列表数据格式的读取需要指定下标(下标从0开始)
