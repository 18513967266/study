Web自动化测试课程

单选框、复选框、文件上传操作

目标

1. 掌握单选框的操作方法
2. 掌握复选框的操作方法
3. 掌握文件上传的操作方法

1. 单选框

说明：单选框就是HTML中<input type="radio" />元素；
      <input type="radio" name="gender" value="1">男
      <input type="radio" name="gender" value="0">女
要选中某个单选框，只需要定位到指定元素之后，执行click()操作即可。

1.1 单选框操作

需求：

打开注册页面A，操作‘水果’下的单选框，首先选中‘西瓜A’，然后依次点击所有的单选框，每点击一次暂停2秒

示例代码

from selenium import webdriver
import time

# 创建浏览器的驱动对象
driver = webdriver.Firefox()

# 打开注册A.html
driver.get("D:\\webAutoTest\\page\\注册A.html")

# 选中‘西瓜A’
driver.find_element_by_id("xga").click()
time.sleep(3)

# 获取所有单选框按钮，并依次点击
fruit_list = driver.find_elements_by_name("fruit")
for fruit in fruit_list:
    fruit.click()
    time.sleep(2)

# 关闭驱动对象
driver.quit()

2. 复选框

说明：复选框就是HTML中<input type="checkbox" />元素；
      <input type="checkbox" name="hobby" value="gw">购物
      <input type="checkbox" name="hobby" value="ly">旅游
      <input type="checkbox" name="hobby" value="yy">音乐
要选中某个复选框，首先要先判断复选框的状态，如果是未选中的状态，只需要执行click()操作即可。

2.1 复选框操作

判断复选框的状态

element.is_selected()   ---> 获取选中状态，选中返回True，未选中返回False

需求：

打开注册页面A，操作‘爱好’下的复选框，选中所有的爱好选项。

示例代码

import time
from selenium import webdriver

# 创建浏览器的驱动对象
driver = webdriver.Firefox()

# 打开注册A.html
driver.get("D:\\webAutoTest\\page\\注册A.html")

# 获取所有复选框按钮，选中所有的复选框
hobby_list = driver.find_elements_by_name("hobby")
for hobby in hobby_list:
    if not hobby.is_selected():
        hobby.click()
        time.sleep(2)

# 关闭驱动对象
driver.quit()

3. 文件上传

说明：文件上传组件就是HTML中<input type="file" />元素；
      <input type="file" name="upfile" />
要实现上传文件的操作，只需要获取到该元素对象，然后调用send_keys(file_path)方法输入文件路径。

3.1 文件上传操作

上传文件的方法

element.send_keys(file_path)   ---> file_path为上传的文件路径

需求：

打开注册页面A，在文件上传框中上传一个本地文件。

示例代码

import time
from selenium import webdriver

# 创建浏览器的驱动对象
driver = webdriver.Firefox()

# 打开注册A.html
driver.get("D:\\webAutoTest\\page\\注册A.html")

# 上传文件
driver.find_element_by_css_selector("[type='file']").send_keys("D:\\note.txt")

time.sleep(3)

# 关闭驱动对象
driver.quit()
